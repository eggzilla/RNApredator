Accession Numbers:

e.g.: NC_000913 for E.coli K-12 MG1655 chromosome
Represent chromosomes or plasmids. If it is in the database an entry will appear in the "Confirm Genome" field. Upon confirmation the selected genome will be searched for interaction with the provided sRNA.

Taxonomy ID:

e.g.: 511145 for E.coli K-12 MG1655
Enter a Taxonomy ID(tax-id) representing a bacterial species. If
it is in the database an entry will appear in the "Confirm Genome" field. Upon confirmation all plasmids and chromosomes associated to the tax-id will be searched for interaction with the provided sRNA.

sRNA:

Enter a sRNA (up to 500 letters) containing only following letters: A,U,G,C,T,a,u,g,c,t. T and t will be interpreted as U for the computation. By using fasta like headers you can enter up to 5 sequences at once. e.g: >test1 AAAAAAAAAAAAAAAAAAAAA >test2 TTTTTTTTTTTTTTTTTTTTT Uploaded sequence files can have up to 2000 letters.

NCBI-Accession Number

OR

NCBI-Taxonomy ID

OR

Select from Available Genomes
